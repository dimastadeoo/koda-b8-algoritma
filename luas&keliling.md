# Algoritma
## Hitung Luas dan Keliling Lingkaran

Algoritma ini digunakan untuk menghitung luas dan keliling lingkaran

1. Mulai
2. masukkan nilai Jari - jari lingkaran
3. menentukan nilai pi: jika jari jari nilainya ketika dibagi 7 sisa pembagian adalah 0 gunakan nilai pi = 22 dibagi 7
4. selain itu gunakan nilai pi = 3.14
5. Hitung luas dengan rumus: pi dikalikan Jari jari kuadrat
6. Hitung Keliling dengan rumus: 2 dikalikan pi dikalikan jari jari
7. selesai
